# hdns

hdns 是运行魔豆上的高级 DNS，它用来对魔豆上的 DNS 进行高级的定制

# 安装

在魔豆上所有的应用都存储在 `/data/apps` 下，所以所有的 hdns 的文件均存放在 `/data/apps/hdns` 里

# 使用

启动 hdns 方法非常简单

	/data/apps/hdns/sbin/hdns.sh restart

# 配置文件

所有的配置文件均存放在 `/data/apps/hdns/conf` 中，如果你有更多好的可供他人使用的 conf 欢迎提交
